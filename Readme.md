Git 컨벤션 정리
- [FEAT] : 새로운 기능을 추가
- [ADD] : 코드, 파일, 폴더 추가
- [FIX] : 버그 수정
- [DOCS] : 문서 수정
- [REFACTOR] : 코드 리펙토링
- [DESIGN] : UI and Texture 변경 및 디자인 변경
- [RENAME] : 파일명|폴더명 수정
- [REMOVE] : 코드, 파일, 폴더의 삭제가 있을 때
- [ETC] : 코드의 짜잘한 수정, 코드 수정 X (파일, 폴더 관리)
